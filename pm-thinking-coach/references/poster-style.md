# Poster/Card Style

Goal: create a polished product-thinking card that is minimalist, orderly, readable, and source-backed. The card should feel like a high-quality strategy memo distilled into one visual page.

## Layout

- Size: 1080 x 1440 portrait.
- Use a strict grid with fixed sections; never let decorative elements overlap text.
- Recommended sections:
  1. Header: topic label, date or product name, title.
  2. Core question: large but controlled, max 3 lines.
  3. Source trail: 3-4 compact source entries with labels and short notes.
  4. Product logic: three equal blocks for product performance, market demand, product value.
  5. Insight: 2-3 concise observations.
  6. Personal reflection: 3 prompts.
- Keep full source links in the accompanying answer; use short source labels on the card.
- Prefer more useful text over illustration. If content is long, reduce type size slightly and truncate gracefully rather than overlap.

## Visual Style

- Minimal editorial card, not a decorative poster.
- Background: off-white or very light warm gray.
- Text: deep ink blue or charcoal.
- Accent: one muted gold/coral/teal line or chip.
- Use thin dividers, small labels, and consistent spacing.
- Avoid large planets, heavy shapes, busy stars, bokeh, gradients, and character illustrations.

## Typography

- Use readable sans-serif defaults for Chinese and English.
- Title 46-56px, question 40-48px, body 24-30px, labels 20-24px.
- Keep line height generous and stable.
- No negative letter spacing.

## Text Card Rules

- Core question: max 52 Chinese characters when possible.
- Source notes: each max 24 Chinese characters.
- Product logic blocks: each max 2-3 lines.
- Insights: each max 34 Chinese characters.
- Prompts: each max 26 Chinese characters.
- If text is longer, preserve meaning and shorten the least important clause.

## Image Generation Prompt Pattern

Use only when the user explicitly wants bitmap art:

```text
Minimal editorial product strategy card background, warm off-white paper, subtle thin line accents, quiet reflective atmosphere, tiny abstract star or paper note motif in corner only, spacious blank areas for Chinese text, no recognizable copyrighted character, no exact book scene, no text in image
```
